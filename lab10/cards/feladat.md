# Feladat

Készíts egy HTML oldalt és egy CSS stíluslapot az alábbi kártyalap elrendezés megjelenítéséhez:

![](cards.png)

A kártyalapok mérete pontosan 2.5 × 3.5 inch kell, hogy legyen. A [Flexbox](https://developer.mozilla.org/en-US/docs/Learn/CSS/CSS_layout/Flexbox) elrendezést használó megoldást adj. A kívánt elrendezés kialakításához szükséges a feladathoz a [`transform`](https://developer.mozilla.org/en-US/docs/Web/CSS/transform) CSS tulajdonság és a [`rotate()`](https://developer.mozilla.org/en-US/docs/Web/CSS/transform-function/rotate) transzformációs függvény.

A kártyalapok megjelenítéséhez használd az [itt](https://unicode.org/charts/PDF/U2600.pdf) található Unicode kártyaszimbólumokat.
